La ciencia de datos o conocida comúnmente como Data Science es una ciencia multidisciplinaria que involucra conocimientos de matemáticas, estadística y ciencias de la computación.

La persona que aplica la Data Science es conocida como científico de datos. El científico de datos utiliza diversas herramientas a su alcance, así como análisis estadísticos de datos, modelos predictivos y visualización de datos, para poder encaminar de manera efectiva la toma de decisiones.

Vid 1.2

Entre las principales actividades para un científico de datos se encuentra la creación de modelos predictivos y la visualización de datos. Nosotros como científicos de datos utilizaremos el software Python, el cual nos permite leer datos y crear a partir de ellos modelos predictivos y gráficas de gran calidad.

Para nuestro curso no es necesario que aprendas todos los detalles de este lenguaje, pero eventualmente te irás familiarizando con sus funciones más comunes.

La librería Pandas te permitirá utilizar funciones específicas para manipular estructuras de datos que se presentan de manera tabular, estas estructuras toman el nombre de data frames. Lo que es una serie de tiempo en realidad es una extracción de una columna
de un conjunto de datos, pero puede servir para representar cosas a través
del tiempo.

Una serie de tiempo es el conjunto de datos estadísticos observados y recopilados durante intervalos regulares de tiempo. La información puede ser registrada de forma diaria, semanal, semestral o en un intervalo definido previamente por el investigador. En esta libreta aprenderás a trabajar con series de tiempo utilizando la librería Pandas.

Una serie de tiempo es una secuencia de datos, observaciones o valores, medidos en determinados momentos y ordenados cronológicamente
